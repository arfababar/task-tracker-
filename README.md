# Task Tracker App

A simple React task tracker with JSON server backend.  
Features:  
- Add new tasks  
- Delete tasks  
- Toggle reminders  
- Basic routing with React Router  

### Run locally:
1. Clone repo  
2. Run `npm install`  
3. Run `npm run dev` for frontend  
4. Run `npx json-server --watch db.json --port 5000` for backend
